# PizzaInBox

## Objetivo
Este projeto tem como objetivo criar um sistema de pizzaria WEB com o intuito de praticar e aprender tecnologias como HTML5, CSS3, JavaScript, etc.

## Escopo

 - Autocadastro de usuário
 - Sistema de pedidos 
 - Homepage
